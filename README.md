###mdserver(mac版)
mac上高度可定制的PHP开发环境,集成必要的扩展,方便使用。
(pkg安装方式),安装方便,是你Mac上的PHP开发利器。


### 文件说明
- host(修改hosts命令)
- mdserver(主功能)
- ss(root启动[未使用])
- Screenshot(截图)
